到訪資料卡產生器

不想使用政府的行蹤記錄軟體，但又想出外用餐？那麼這個工具就適合你！

只要輸入姓名、聯絡電話，系統就會產生一張到訪資料卡，讓你隨時列印使用。

※此工具只在裝置端進行到訪資料卡的產生，不須使用任何網路連線。此外，此工具的原始碼已放置在 GitHub 上供查閱。

負責人注意事項
  • 本到訪資料卡只作記錄到訪資訊用。
  • 在有確診人士到訪時，方可把此到訪資料卡交予衛生防護中心作個案追蹤之用。
  • 本到訪資料卡須在卡面記載的到訪日期起計 31 天後銷毀。
